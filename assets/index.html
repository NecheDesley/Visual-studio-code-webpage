Meet IntelliSense

Go beyond syntax highlighting and autocomplete with intellisense, which provides smart completions based on variable types, function definitions and imported modules.

Print statement debugging is a thing on the past.

Debug code right from the editor. launch or attach to your running apps and debug with breaks points, call stacks and an interactive console.

Git commands built-in

Working with git and other SCM providers has never been easier. Review diffs, stages files and makes commits right from the editor. Push and pull from any hosted SCM service.

Extensinle and customizable

Want even more features? install extensions to add new languages, themes, debuggers and to connect to additional services. Extensions run in separate processes, ensure they won't slow down your editor, all from within VS Code.

Deploy with confidence and ease

With Microsoft Azure you can deploy and host your React, Angular, Vue, Node, Python (and more!) sites, stores and query relational and document based data, and scale with serverless computing, all with ease,all from within VS Code.
